# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Open-source community health files: `CODE_OF_CONDUCT.md`, issue templates, and `CHANGELOG.md`.
- Continuous integration workflow for lint, build, and test on pull requests.
- Expanded `CONTRIBUTING.md` with branch, commit, and code-style conventions.

### Changed

- Polished `README.md` with badges, corrected clone URL, and a "Getting Help" section.
- Added `repository`, `bugs`, `homepage`, and `engines` metadata to `package.json`.

## [0.1.0] - 2026-06-16

### Added

- Initial public release of MyFamilyButler.
- Next.js 16 app-router dashboard for family calendar management.
- Multi-channel messaging: WhatsApp (Meta Cloud API), Telegram Bot API, and 360dialog.
- AI-powered natural-language event extraction with Gemini primary and OpenAI fallback.
- Vision processing for extracting events from photos and school letters.
- Google Calendar bidirectional sync.
- Supabase-backed authentication, magic-link login, and family sharing.
- AI-assisted delivery model with multi-role review gates.
